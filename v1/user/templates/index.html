APIs For Food Order App
Authentication
  • User Sign Up : v1/auth/register
  • User Sign In : v1/auth/login
Register A Cafe
  • Register : v1/shop/add
  • Form Input : name, address
  • Token Input : cafe [user_id]
Item Category
  • Category List : v1/category
  • Category Detail : v1/category/id
  • Category Create : v1/category/add
  • Form Input : name, address
  • Token Input : user [user_id], cafe [cafe_id]
  • Category Update : v1/category/id/edit
  • Category Delete : v1/category/id/delete
Items
  • Item List : v1/item/id/delete
  • Item Detail : v1/item/id
  • Item Create : v1/item/add
  • Item Update : v1/item/id/edit
  • Item Delete : v1/item/id/delete
